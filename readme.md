# Quick Attendance Program
`*******  Release : V1.0  ********

An program to Quickly take attendance by roll numbers

This C program is designed to take upto 100 numbers, after which it outputs the data and asks to save if needed.

## Base functions
* Take attendance of upto 100 roll numbers
* Exit anywhere (Early quit)
* show rollnumbers neatly

## Extra functions
* 2 types of output formats
* Save the program output to a file
  * User must provide filename

## Planned

  * Add optional absent numbers below present group
